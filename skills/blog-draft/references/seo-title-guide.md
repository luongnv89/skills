# SEO Title Optimization Guide

## Core Rules

1. **Primary keyword near the start** - Place main keyword in first 3-5 words
2. **Accurate description** - Title must match content; avoid clickbait
3. **Length: 45-65 characters** (~550 pixels) to avoid truncation
4. **One keyword per post** - Prevent competing titles across your site
5. **Human-readable first** - Never stuff keywords

## Click-Through Rate Boosters

- Add specific benefit: "...to Boost Organic Traffic", "...That Actually Convert"
- Use numbers: "7 Ways to...", "10 Tips for..."
- Action verbs: "Boost", "Learn", "Master", "Discover"
- Brackets for context: "(2026 Guide)", "[Checklist]", "(With Examples)"
- Emotional/curiosity triggers (keep promises realistic)

## Title Patterns

### Pattern 1: Keyword + Benefit
```
SEO Title Tags: Master Optimization in 5 Steps
```

### Pattern 2: Number + Adjective + Keyword + Outcome
```
7 Proven SEO Title Tips to Increase CTR
```

### Pattern 3: How to + Keyword + Outcome
```
How to Write SEO-Friendly Blog Titles for Better Rankings
```

### Pattern 4: Keyword + Audience + Year
```
SEO Blog Title Examples for Beginners (2026 Guide)
```

### Pattern 5: Keyword + Brand (with separator)
```
SEO Blog Title Formulas That Get Clicks | YourBrand
```

## Title Generation Workflow

1. Identify primary keyword matching search intent
2. Generate 5 title options using patterns above
3. Check length (45-65 chars) with SERP simulator
4. Select best balance of keyword placement, clarity, and click appeal

## Quick Checklist

- [ ] Primary keyword in first 5 words?
- [ ] Length under 65 characters?
- [ ] Includes specific benefit or outcome?
- [ ] Human-readable and natural?
- [ ] Matches actual content?
- [ ] Uses number, bracket, or action verb?
