# SEO Content Optimization Guide

## Keyword Placement

### Primary Keyword
- Include in **first 100 words** (ideally first paragraph)
- Use in **at least one H2 heading**
- Include in **conclusion**
- Target **1-2% density** (e.g., 10-20 times in 1000 words)

### Keyword Variations
- Use synonyms and related terms naturally
- Include long-tail variations in subheadings
- Avoid exact-match repetition (sounds unnatural)

## Content Structure

### Headings (H2, H3, H4)
- Use **one H1** only (the title)
- H2s for main sections (include keyword in 1-2)
- H3s for subsections (use keyword variations)
- Keep hierarchy logical (don't skip levels)

### Paragraphs
- **2-4 sentences max** per paragraph
- One idea per paragraph
- Use transition words between sections

### Scannable Elements
- Bullet points for lists (3+ items)
- Numbered lists for steps/rankings
- Bold for key terms and takeaways
- Tables for comparisons
- Blockquotes for important quotes

## Meta Elements

### Meta Description
- **150-160 characters**
- Include primary keyword
- Add a call-to-action
- Make it compelling (it's your ad in search results)

Example:
```
Learn 7 proven SEO title formulas that boost click-through rates.
Includes templates and real examples. Start ranking higher today.
```

### URL Slug
- **3-5 words** maximum
- Include primary keyword
- Use hyphens (not underscores)
- Lowercase only
- Remove stop words (a, the, and, of)

Good: `seo-title-formulas-ctr`
Bad: `7_proven_seo_title_formulas_that_will_help_you_boost_your_ctr`

## Links

### Internal Links
- Link to **2-3 related posts** minimum
- Use descriptive anchor text (not "click here")
- Link early in content when relevant
- Helps distribute page authority

### External Links
- Link to **authoritative sources** (studies, official docs)
- Opens credibility and trust signals
- Use 2-5 external links per post
- Ensure links open in new tab for UX

## Images

### Alt Text
- Describe the image accurately
- Include keyword if natural
- Keep under 125 characters
- Don't start with "Image of..."

Good: `SEO title tag formula infographic showing 5 patterns`
Bad: `image1.jpg` or `SEO SEO SEO title SEO`

### File Names
- Descriptive, keyword-rich
- Use hyphens: `seo-title-formulas.png`

## Featured Snippet Optimization

Target position zero by formatting answers for common queries:

### Paragraph Snippets
- Answer the question in **40-60 words**
- Place directly after the question as H2

### List Snippets
- Use ordered/unordered lists
- 5-8 items ideal
- Start with action verbs

### Table Snippets
- Use for comparisons, specs, pricing
- Clear headers
- Concise cell content

## Readability

### Target Metrics
- **Flesch Reading Ease**: 60-70 (8th-9th grade level)
- **Sentence length**: 15-20 words average
- **Passive voice**: Under 10%

### Tips
- Use active voice
- Choose simple words over complex
- Break up long sentences
- Read aloud to check flow

## Quick Optimization Checklist

```
[ ] Primary keyword in first 100 words
[ ] Keyword in 1-2 H2 headings
[ ] 1-2% keyword density (natural)
[ ] Meta description: 150-160 chars with keyword + CTA
[ ] URL slug: short, hyphenated, keyword-rich
[ ] 2-3 internal links with descriptive anchors
[ ] 2-5 external links to authoritative sources
[ ] All images have descriptive alt text
[ ] Short paragraphs (2-4 sentences)
[ ] Scannable: bullets, numbers, bold, tables
[ ] Featured snippet: direct answer for main question
[ ] Readability: 8th-9th grade level
```
